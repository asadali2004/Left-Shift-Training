## How to Use:

Each question folder contains:
- `.csproj` - Project file
- `Program.cs` - Main program with TODO comments
- Model classes (with properties defined)
- Manager/Utility class (with method signatures)
- `Question.md` - Detailed requirements

## Getting Started:

1. Open a question folder
2. Read `Question.md` for requirements
3. Implement the TODO sections
4. Follow the sample use cases
5. Test your implementation
