# StudentApp - Delegates and Result Processing

## Problem Statement
Build a console application to manage a list of students and evaluate their results.

## Requirements
- Create a `Student` model with `StudentId`, `StudentName`, and `StudentMark`.
- Store at least 5 students in a list.
- Display all students.
- Sort students by marks in descending order and display them.
- For each student, print **Pass** if marks are >= 40; otherwise **Fail**.
- Demonstrate use of `delegate`, `Action`, `Func`, and `Predicate`.

## Expected Output (Sample)
```
Student's Id: 101, Student's Name: Asad, Student's Mark: 89
...
Asad: Congratulations!!! You Pass
...
```
